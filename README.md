# Московское метро — трекер посещений

Статический сайт на Jekyll для отслеживания посещённых станций Московского метро.

## Структура проекта

```
Metro/
├── _config.yml              # Конфигурация Jekyll
├── _layouts/
│   ├── default.html         # Базовый шаблон (шапка, подвал)
│   └── station.html         # Шаблон страницы станции
├── _stations/               # Markdown-файлы станций (коллекция Jekyll)
│   └── линия-название.md    # Один файл = одна станция
├── assets/
│   ├── css/
│   │   └── style.css        # Все стили
│   └── photos/              # Фотографии
│       └── название-станции/
│           ├── cover.jpg    # Титульное фото
│           └── *.jpg        # Остальные фото
├── index.html               # Главная страница (сетка всех станций)
├── lines.html               # Станции, сгруппированные по линиям
└── Gemfile
```

## Добавление новой станции

1. Создайте файл в папке `_stations/`, например `_stations/koltsevaya-kurskaya.md`
2. Заполните front matter (YAML-шапку):

```yaml
---
title: "Курская"
visited: true                    # true / false
visited_date: 2024-05-20        # дата посещения

line_name: "Кольцевая"
line_number: 5
line_color: "#8d3f2b"

cover_photo: /assets/photos/kurskaya/cover.jpg

photos:
  - url: /assets/photos/kurskaya/cover.jpg
    caption: "Описание фото"
  - url: /assets/photos/kurskaya/hall.jpg
    caption: "Центральный зал"

architect: "Автор"
opened: 1950
depth_m: 33
---
```

3. Напишите текст описания в формате Markdown ниже шапки
4. Добавьте фотографии в папку `assets/photos/название-станции/`

## Запуск локально

```bash
bundle install
bundle exec jekyll serve
# Открыть: http://localhost:4000
```

## Сборка для публикации

```bash
bundle exec jekyll build
# Результат в папке _site/
```
